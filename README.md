## Install

1. In the `root` directory:

```git
$ git clone https://github.com/ochukai/hexo-theme-ochuunn.git themes/ochuunn
```

2. Change file `_config.default.yml` to `_config.yml`;

3. Change the `theme` property in the `config.yml` file.

```yml
# theme: landscape
theme: ochuunn
```
